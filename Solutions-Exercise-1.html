Question 1

  1. What does the df.info() method tell you about the gapminder dataset?

Solution

Click to toggle answer The df.info() method provides information about the number of rows and columns, the count of non-null entries in each column, the data types of each column, and the memory usage of the DataFrame.
Show code cell source Hide code cell source 
import pandas as pd

# URL to the raw CSV file on GitHub
url = 'https://raw.githubusercontent.com/kirenz/datasets/master/gapminder.csv'

# Read the CSV file into a DataFrame
df = pd.read_csv(url)

df.info()

Question 2

  1. How many rows and columns are present in the gapminder dataset? Use an appropriate attribute to find out.

Solution

Click to toggle answer The shape attribute (df.shape) returns a tuple representing the dimensionality of the DataFrame. The first element of the tuple is the number of rows, and the second element is the number of columns. 
df.shape

(1704, 6)

Question 3

  1. List all the column names of the gapminder dataset.

Solution

Click to toggle answer The columns attribute (df.columns) returns the column labels of the DataFrame. 
df.columns

Index(['country', 'continent', 'year', 'lifeExp', 'pop', 'gdpPercap'], dtype='object')
